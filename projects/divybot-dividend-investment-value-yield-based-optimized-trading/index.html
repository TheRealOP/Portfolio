DIVYBOT (Dividend, Investment, Value, Yield, Based, Optimized, Trading)

PythonTrading BotFinancePandasyfinanceAlpaca API

Developed a stock trading bot in Python which used libraries such as pandas, yfinance, alpaca-py, and numpy to calculate and optimize stock distributions based on dividend growth, FCFE, and 200-day SMA comparison. Implemented financial algorithms to assess stock performance and optimize portfolio allocation, incorporating metrics like dividend yield and free cash flow to equity (FCFE). Currently testing using Alpaca API paper trading.
