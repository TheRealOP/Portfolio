High Five Robot

Designed and built a robot that used MediaPipe for gesture recognition to detect high fives using computer vision. Programmed servo controls to trigger a mechanical \"high five\" response when the hand was detected. Utilized Python and the MediaPipe Hands API for real-time hand tracking and gesture recognition. Used Onshape CAD software to design the robotic arm.
